# Structure of Dynamics 365 Finance data sources for FEC

In Dynamics 365 Finance it is possible to generate FEC main file or the following annexes to it:

-	Customers fiscal year opening balances annex 
-	Vendors fiscal year opening balances annex 
-	Customers transactions annex for the period specified
-	Vendors transactions annex for the period specified

It is also possible to generate FEC including customers and vendors fiscal year opening balances. We recommend to use this option for companies with small number of records in the reporting period. Reporting period for this option must include the beginning of the fiscal year.

The following sections list the data sources that are used in FEC main file and annexes.
