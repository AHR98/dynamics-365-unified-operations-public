# Structure of Dynamics 365 Finance data sources for FEC

In Dynamics 365 Finance, it's possible to generate the FEC main file or the following annexes:

-	Customers fiscal year opening balances annex 
-	Vendors fiscal year opening balances annex 
-	Customers transactions annex for a specified period
-	Vendors transactions annex for a specified period

An FEC that includes customers and vendors fiscal year opening balances can also be generated. Use this option for companies with a small number of records in the reporting period. The reporting period must include the beginning of the fiscal year.

The following sections list the data sources that are used in FEC main file and annexes.
